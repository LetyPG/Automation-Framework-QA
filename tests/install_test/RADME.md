
Documento para explicar el modulo, en este caso se trata de softeware UI, pero si el software desarrolla una version de ap0licativo para escritorio o movil, esta suite podria contener la logica que evalue la correcta instalacion en diferenetes sistemas operativos.

Para esto se necesitaria crear entornos virtuales con diferenete OS y ejecutar el test en cada uno de ellos.


Prereqisitos:

Tener instalado o tenr conocimeinentos en el manejo de herramientas de despligue de entornos virtuales, como por ejemplo:

- Vagrant
- Docker
- AWS
- Azure


Luego se requiere crear un modulo que contenga la logica de provisionamiento de para el software:

- Instalacion de paquetes y dependencias
- Configuracion de variables de entorno
- Configuracion de permisos
- Configuracion de servicios
- Configuracion de base de datos
- Configuracion de aplicacion
- Configuracion de entorno de pruebas
- Configuracion de entorno de produccion
